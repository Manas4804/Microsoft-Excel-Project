# Microsoft Excel Project

This repository is a small Excel workbook collection centered on a bike sales analytics dashboard. The main workbook combines source data, pivot-style summaries, slicers, and charts to make the sales patterns easy to explore without writing any code.

## What’s Included

- `Bike Sales Dashboard.xlsx` - the main dashboard workbook shown in the preview image.
- `Excel Charts Tutorial File.xlsx` - a companion file for chart practice and charting examples.
- `Formula Excel Template.xlsx` - a workbook for formula practice and reusable calculation patterns.
- `Conditional Formatting Excel Tutorial File.xlsx` - examples of highlighting data with conditional formatting.
- `Data Cleaning Excel Tutorial.xlsx` - practice file for cleaning and preparing messy data.
- `XLOOKUP Excel Tutorial File.xlsx` - lookup formula practice workbook.
- `Sales.xlsx` - an additional sales-related workbook included with the project.
- `Excel Project Dataset.xlsx` - the dataset workbook used to support the dashboard and tutorials.

## Dashboard Overview

The main dashboard is designed to answer a few common bike sales questions quickly:

- Which customer groups have the highest average income per purchase
- How age brackets differ between buyers and non-buyers
- How commute distance relates to bike purchases
- How filters such as marital status, region, and education change the story

The dashboard uses interactive slicers on the left side to filter the visuals by customer attributes. The charts update together so you can compare segments consistently.

### Key Visuals

- **Avg Income Per Purchase** - compares average income by gender and bike purchase status.
- **Customer Age Brackets** - shows how buyers and non-buyers distribute across age groups.
- **Customer Commute** - compares bike purchase counts across commute-distance buckets.
- **Slicers** - let you filter by marital status, region, and education.
